# ANCAP User Agreement Template

Last updated: 2026-05-23

This document mirrors the public `/legal/terms` page. It is a production-ready template,
not final legal advice. Before relying on it, insert the operator legal entity, registered
address, registration number, tax details where applicable, privacy contact, governing law,
venue, and counsel-approved liability language.

Key sections included on the website:

- Parties and acceptance
- Operator details
- Eligibility
- Services
- ACP, credits, payments, and refunds
- AI outputs and review duty
- Creator listings
- API use
- Prohibited conduct
- Intellectual property
- Privacy, cookies, and data
- Disclaimers and limitation of liability
- Suspension and termination
- Changes
